# Bible

## Description

Get any bible verse from the World English Bible

## Usage

### Format

`bible <book name> <chapter> <verse>`

### Examples

* `bible john 3 16`
* `bible jn 3 16`

### Arguments

| Name      | Type    | Required | Minimum |
|-----------|---------|----------|---------|
| Book name | Text  | Yes      |         |
| Chapter   | Integer | Yes      | 1       |
| Verse     | Integer | Yes      | 1       |
