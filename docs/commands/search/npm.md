# NPM

## Description

Get information about an NPM package

## Aliases

* `npm`
* `npmsearch`

## Usage

### Format

`npm-search <package name> [show full results]`

### Examples

* `npm-search ms`
* `npm-search commando-provider-keyv`

### Arguments

| Name              | Type    | Required | Minimum | Maximum |
|-------------------|---------|----------|---------|---------|
| Package name      | String  | Yes      |         |         |
| Show full results | True/False | No       |         |         |
