# Get Minecraft skin

## Description

Get the skin of a Minecraft user.

## Aliases

* `get-mc-skin`
* `getminecraftskin`
* `getmcskin`

## Usage

### Format

`get-minecraft-skin <username>`

### Examples

* `get-minecraft-face Notch`

### Arguments

| Name     | Type   | Required |
|----------|--------|----------|
| Username | String | Yes      |
