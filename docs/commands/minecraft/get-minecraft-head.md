# Get Minecraft head

## Description

Shows an isometric view of a Minecraft user's head.

## Aliases

* `get-mc-head`
* `getminecrafthead`
* `getmchead`

## Usage

### Format

`get-minecraft-head <username>`

### Examples

* `get-minecraft-head Notch`

### Arguments

| Name     | Type   | Required |
|----------|--------|----------|
| Username | String | Yes      |
