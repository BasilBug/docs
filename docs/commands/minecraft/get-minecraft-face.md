# Get Minecraft face

## Description

Shows a front view of a Minecraft user's face.

## Aliases

* `get-mc-face`
* `getminecraftface`
* `getmcface`

## Usage

### Format

`get-minecraft-face <username>`

### Examples

* `get-minecraft-face Notch`

### Arguments

| Name     | Type   | Required |
|----------|--------|----------|
| Username | String | Yes      |
