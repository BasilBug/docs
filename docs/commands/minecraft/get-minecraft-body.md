# Get Minecraft body

## Description

Shows a Minecraft user's body.

## Aliases

* `get-mc-body`
* `getminecraftbody`
* `getmcbody`

## Usage

### Format

`get-minecraft-body <username>`

### Examples

* `get-minecraft-body Notch`

### Arguments

| Name  | Type | Required | Minimum | Maximum |
|-------|------|----------|---------|---------|
| Username | String |Yes |  |  |
