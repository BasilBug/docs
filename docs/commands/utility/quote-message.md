# Quote message

## Description

Quote a message from a server.

## Aliases

* `quote`
* `quotemessage`

## Usage

### Format

`quote-message <message ID>`

### Examples

* `quote-message 424936127154094080`

### Arguments

| Name       | Type    | Required |
|------------|---------|----------|
| Message ID | Message | Yes      |
